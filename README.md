# PokeShop

Site e-commerce de cartes Pokemon, built with Next.js 14, Tailwind CSS et Stripe.

## Lancer en local

1. npm install
2. Copier .env.example en .env.local et remplir les cles Stripe
3. npm run dev
4. Ouvrir http://localhost:3000

## Deployer sur Vercel

1. Pousser sur GitHub
2. Importer sur vercel.com
3. Ajouter les variables d'environnement
4. Deployer
